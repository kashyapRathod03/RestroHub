## 🎯 About

RestroHub is an all-in-one digital solution designed specifically for Indian restaurants. From street-side dhabas to fine dining establishments, we help restaurants create digital menus, accept payments, sync with delivery platforms, and manage their online presence.

### Why RestroHub?

| Problem | Our Solution |
|---------|--------------|
| 🇮🇳 Not India-focused | Built for Indian restaurants (Thali menus, regional languages) |
| 📱 Complex setup | 5-minute QR menu creation |
| 💳 Payment integration hassles | Direct UPI payment links |
| 🌐 Language barriers | AI-powered translation in 25+ languages |
| 📦 Manual aggregator updates | Auto-sync with Zomato & Swiggy |

## 🚀 Getting Started

### Prerequisites

```bash
Make sure you have installed:

Java 21
Gradle
PostgreSQL
Git
IDE (IntelliJ IDEA recommended)
Lombok & MapStruct annotation processing enabled

```

### Installation

```bash
# Clone the repository
git clone https://github.com/rdodiya/RestroHub.git

# Navigate to project directory
cd RestroHub

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env

# Start development server
npm run dev
```

### Environment Variables

```env
# ===================
# Core Configuration
# ===================
NODE_ENV=development
PORT=3000

# ===================
# Database
# ===================
MONGODB_URI=mongodb://localhost:27017/restrohub
REDIS_URL=redis://localhost:6379

# ===================
# Authentication
# ===================
JWT_SECRET=your-secret-key
NEXTAUTH_URL=http://localhost:3000
NEXTAUTH_SECRET=your-nextauth-secret

# ===================
# File Storage
# ===================
AWS_ACCESS_KEY_ID=your-aws-key
AWS_SECRET_ACCESS_KEY=your-aws-secret
AWS_S3_BUCKET=restrohub-uploads

# ===================
# Payment Integration
# ===================
RAZORPAY_KEY_ID=your-razorpay-key
RAZORPAY_SECRET=your-razorpay-secret

# ===================
# WhatsApp Integration
# ===================
WHATSAPP_API_KEY=your-whatsapp-api-key
WHATSAPP_BUSINESS_ID=your-business-id
WHATSAPP_PHONE_NUMBER_ID=your-phone-id

# ===================
# Translation API
# ===================
GOOGLE_TRANSLATE_API_KEY=your-translate-key

# ===================
# Aggregator APIs
# ===================
ZOMATO_API_KEY=your-zomato-key
ZOMATO_RESTAURANT_ID=your-zomato-restaurant-id
SWIGGY_API_KEY=your-swiggy-key
SWIGGY_RESTAURANT_ID=your-swiggy-restaurant-id
```

### 🚀 How to Contribu  te

```bash
# 1. Fork the repository
# 2. Clone your fork
git clone https://github.com/rdodiya/RestroHub.git

# 3. Create a feature branch
git checkout -b feature/whatsapp-integration

# 4. Make your changes
# 5. Run locally and test it

./gradlew clean build
./gradlew bootRun
Application will start at: http://localhost:8181/restroly
Example : http://localhost:8181/restroly/api/v1/foods/{id}

# 6. Commit with conventional commits
git commit -m "feat: add WhatsApp status update service"

# 7. Push to your fork
git push origin feature/whatsapp-integration

# 8. Open a Pull Request
```

## 📘 Swagger API Documentation

Swagger UI is available at:

```
http://localhost:8181/restroly/swagger-ui.html
```

> If Swagger URLs look incorrect, ensure `server.servlet.context-path=/restroly`
> and OpenAPI `@OpenAPIDefinition(servers = "/restroly")` is configured.

